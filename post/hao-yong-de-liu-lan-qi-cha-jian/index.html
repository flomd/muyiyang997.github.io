好用的浏览器插件

2020-12-01 5 min read

1、Tampermonkey
最强大的脚本插件，俗称"油猴"，在火狐浏览器上叫“Greasemonkey”，可以为你的浏览器加载各种各样的脚本，对你正在访问的指定网站进行修改，屏蔽掉一些烦人的东西。比如把某一块的字体颜色改掉，添加一个功能按钮、直接下载百度网盘文件、去掉视频播放广告以及绕过搜索引擎的跳转提示等等。

2、Internet Download Manager
IDM 是一款非常经典老牌、功能强大的 Windows 文件多线程下载加速软件，被称为 HTTP 下载神器！除了下载加速之外，它还提供了自动链接捕获、下载队列、站点抓取、映射服务器、视频音频下载、国外网盘加速下载、静默下载等多种功能。

3、LastPass
LastPass是一款屡次获奖的密码管理器插件，除了可以在浏览器中作为插件使用，还可以在Windows、Mac、Linux 三大系统安装桌面端使用，同时iOS、Android等移动平台也可以使用。通过自动云同步，用户只需记住一个主密码，就可以随时随地、不受设备数量限制地享受密码自动填充、安全笔记、好友密码共享等功能。

4、Adblock Plus
Adblock Plus是一款非常强大的广告拦截插件，可免费使用，支持包括Chrome、Firefox等多种浏览器。安装并启用该插件以后，可自动屏蔽掉网页中的广告并把空白的页面合并到一起，同时它也可以禁用广告跟踪、拦截恶意软件、屏蔽社交网站等。

5、Infinity 新标签页
Infinity新标签页是一款基于html5的免费浏览器插件，支持Chrome和火狐浏览器。可一步添加常用网站到新标签页并以图标展示，可自定义搜索引擎，精选高清壁纸，还有天气、笔记、书签、待办事项、历史记录、Gmail提醒等小挂件帮助我们更高效的使用浏览器。

6、书签侧边栏
书签侧边栏是一款Chrome浏览器书签管理插件，可在浏览器视图区两侧添加固定式书签栏。支持对侧边书签栏的字体、配色方案、文字颜色，甚至鼠标悬停颜色等进行自定义更改。支持对书签栏的整体大小、宽高、图标、缩进进行设置，自由度非常高。相比与Chrome自带的书签栏，该书签栏更加美观简洁、实用。

7、FireShot
一个非常适合网页截全图的Chrome插件，可以截取各种形式的图片，整张截图、可视部分的、选定区域截图，也可以选择多种输出方式。同时可以有涂鸦、编辑截取的图片的功能，高级版本拥有强大的标记注释功能，绝对是新媒体小编们的必备利器！

8、新媒体管家
非常好用的运营管理工具，可一键管理你的所有新媒体帐号，包括微信公众平台、今日头条等11家新媒体平台。提供有微信公众号排版功能，可直接在微信公众号的后台完成文章的找图、修图、编辑、排版等工作，可大大提升工作效率。除了排版外，还支持热点收集、数据分析、链接生成、图文批量删除等等。

9、Fatkun图片批量下载
Fatkun 图片批量下载是一款免费又好用的 Chrome 浏览器图片下载插件，可帮你快速找出当前页面上的全部图片，而且可以根据图片分辨率、链接等对图片进行筛选，也可以自由点选，最后一键保存所有的图片。

10、简悦
一款非常好用的沉浸式阅读插件，在阅读模式下，可生成符合中文阅读习惯 的页面， 零干扰，沉浸式阅读；在聚焦模式，不改变当前页面的结构，仅仅高亮需要阅读的部分，适合临时阅读；除此之外，还可以自由选择主题，自定义文字间距、行距等等。

11、掘金
掘金插件是一款为设计师、程序员、产品经理每日发现优质内容的chrome插件，是一个内容非常丰富的社区，社区内有前端、后端、iOS、Android、人工智能、开发工具等多个板块，产品经理、开发者可以在这里找到最新的开发资讯，最前沿的技术干货。
